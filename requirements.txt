# GPU NOTES (native Windows):
#   TensorFlow >= 2.11 dropped CUDA support on native Windows.
#   This codebase requires TF >= 2.12 (uses standalone keras.saving API),
#   so TF 2.10 is NOT a compatible downgrade.
#
#   Two options for GPU on native Windows:
#
#   Option A — WSL2 (recommended by TF):
#     Install WSL2, then inside WSL2:
#       pip install -r requirements.txt
#     CUDA is fully supported inside WSL2 with TF >= 2.12.
#
#   Option B — DirectML plugin (native Windows, DirectX 12 GPU):
#     pip install tensorflow tensorflow-directml-plugin
#     Supports AMD, Intel, and NVIDIA GPUs via DirectX 12.
#     See: https://github.com/microsoft/tensorflow-directml-plugin
#
